## Solution `java`

```java
import java.util.*;

public class Main {

    public static void main(String[] args) {
        // Write your code here
        System.out.println("*****");
        System.out.println("   *");
        System.out.println("  *");
        System.out.println(" *");
        System.out.println("*****");
    }
}
```
